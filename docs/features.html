SavePlus Features

Discover how SavePlus enhances your Maya workflow with intelligent versioning and file management tools

Intelligent Versioning

SavePlus goes beyond simple incrementing by intelligently analyzing your filenames to preserve naming patterns while updating version numbers.

  • Pattern Recognition: Automatically identifies version numbers in various formats and positions
  • Preserves Project Identifiers: Maintains important project codes, artist names, and other identifiers
  • Handles Complex Naming: Works with studio-specific naming conventions and pipeline requirements
  • Version Preview: Shows you what your next filename will be before saving

Examples of Intelligent Versioning

  • character.ma → character02.ma
  • character5.ma → character6.ma
  • character_v001.ma → character_v002.ma
  • J02_Smith_John_blocking_wip_01.ma → J02_Smith_John_blocking_wip_02.ma
  • scene45_99.ma → scene45_100.ma

Comprehensive Version Tracking

Keep a complete record of your file history with SavePlus's robust version tracking system.

  • Version Notes: Document changes with each save to maintain a detailed work log
  • Quick Notes: Rapidly add context without interrupting your workflow
  • File Groups: Automatically organizes related files together
  • Recent Files List: Quickly access your most recent work across all projects
  • Exportable History: Generate documentation of your version history

Version history is maintained across Maya sessions and provides a searchable record of your work evolution, making it easy to track changes, revert to previous states, or document project progress.

Advanced Name Generator

Maintain consistent file naming across projects and teams with the built-in name generator that follows studio conventions.

  • Structured Templates: Generate standardized filenames based on established patterns
  • Pipeline Integration: Select from predefined pipeline stages to maintain workflow standards
  • Artist Attribution: Include artist information for clear ownership
  • Project Identifiers: Maintain assignment or job codes for organizational clarity

The Name Generator is particularly valuable in team environments where consistent naming is critical for pipeline integration and project management.

Safety Features

SavePlus includes multiple layers of protection to prevent work loss and maintain file integrity

Save Reminders

Never forget to save again with customizable reminders that alert you when it's time to save your work.

  • Configurable time intervals
  • Visual indicator changes color as time passes
  • Non-intrusive notification system

Automatic Backups

Create timestamped backup copies of your work to provide recovery options and prevent data loss.

  • Scheduled automatic backups
  • Manual backup creation
  • Intelligent backup naming

Conflict Resolution

Smart handling of file conflicts to prevent accidental overwrites and data loss.

  • Proactive conflict detection
  • Multiple resolution options
  • Automatic unique naming

Workflow Integration

SavePlus seamlessly integrates with your existing Maya workflow and pipeline

Quick Access

Launch SavePlus instantly with the shelf button or keyboard shortcuts:

  • Ctrl+S: Save Plus (increment save)
  • Ctrl+Shift+S: Save As New
  • Ctrl+B: Create Backup

Reference Path Integration

Extract paths from referenced files to maintain organizational structure and ensure animation files are saved alongside their reference assets.

Project Directory Support

Full compatibility with Maya's project structure, ensuring files are saved in the correct locations within your project hierarchy.

Custom Save Locations

Configure default save paths or browse to specific locations, with visual indicators showing the current save destination.

File Type Selection

Easily toggle between Maya ASCII (.ma) and Maya Binary (.mb) formats based on project requirements.

Customizable Interface

Configure which sections are expanded or collapsed by default for a streamlined experience that matches your workflow needs.

Extensive Customization

Tailor SavePlus to match your specific workflow and preferences.

  • Default Settings: Configure default file types, save locations, and project directories
  • UI Preferences: Control which sections are expanded by default
  • Reminder Intervals: Set the frequency of save reminders
  • Backup Configuration: Enable automated backups and set intervals
  • Version Notes: Toggle version notes for rapid or detailed documentation

All preferences are stored per user and persist between Maya sessions, ensuring a consistent experience tailored to your needs.
